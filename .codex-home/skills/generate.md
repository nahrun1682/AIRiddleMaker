# なぞなぞ生成スキル

## 生成ルール

以下の制約に従って日本語なぞなぞを1問生成してください。

### 必須条件
- 答えが一つに定まること（複数の答えが成立する問題は作り直し）
- 一つの現象から一つの逆説で成立すること（複数ステップの論理は禁止）
- 逆説は二方向あること（例：「食べるほど減るのに、食べないと増える」->答え「食欲」）
- 答えを聞いた瞬間に「なるほど！」と納得できる構造であること

### 作り方の指針
- 概念（秘密・時間・平和など）から作ろうとしない
- 日常の物理現象や身近な事実を観察して作ること
- 例：「寒くなると伸び、暑くなると縮む」→ つらら

### 禁止事項
- 「私は何でしょう」「What am I?」などの定番フォームは2〜3個まで
- よくある答え（影・時間・風・沈黙・炎・記憶・夢）は禁止
- 英語の発想をそのまま日本語にしたものは禁止
- 答えが複数成立するものは禁止


### 日本語固有パターンを活用すること

以下のいずれかを使うこと:

- **paradox**: 逆説・現象系（例: 「食べるほど減るのに、食べないと増える」→ 「食欲」）
- **pun**: 同音異義・ダジャレ系（例: 「食べられないパン → フライパン」）
- **char_extract**: 文字分解系（例: 「機械の真ん中 → 蚊（き**か**い）」）
- **reverse_read**: 逆読み系（例：まるく→車）

- **kanji_structure**: 漢字の部首・構造を使うもの

## 出力フォーマット

以下のJSON形式のみで出力してください:

```json
{"question": "問題文", "answer": "答え", "pattern": "パターン名"}
```
